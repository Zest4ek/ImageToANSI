# <img src="app.ico" alt="icon" width="40"/> ImageToANSI
Простая программа, позволяющая вывести изображение в консоль, используя 216-цветовую палитру.<br>
<img src="https://github.com/trapd00r/win95-winxp_icons/blob/master/icons/w2k_warning.ico" alt="warning_icon" width="25"/> **Примечание:** Конвертируйте изображение в `.png`, `.jpg` или `.jpeg`, чтобы избежать визуальных багов!
